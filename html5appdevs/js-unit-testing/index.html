Unit Testing

Domenic did an amazing job. I should skip this slide.

  • I'm tired of manually refreshing and manually debugging everything

Imperative code is so easy


    function add(){
        var sum = 0;
        for(var i = 0;i<arguments.length;i++){
            sum += arguments[i];
        }
        return sum;
    }
    
    equal(10, add(1, 2, 3, 4));
    equal(100, add(50, 50));

And I can refactor


    function add(){
        return Array.prototype.slice.call(arguments)
                    .reduce(function(prev, current){return prev + current;});
    }
    
    equal(10, add(1, 2, 3, 4));
    equal(100, add(50, 50));
